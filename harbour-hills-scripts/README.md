# Harbour & Hills Care — FB Reply Scripts

Internal tool for posting to Facebook groups.

## Deploy to Vercel (free, ~5 minutes)

1. Create a free account at https://github.com and upload this folder as a new repository
2. Go to https://vercel.com and sign in with your GitHub account
3. Click "Add New Project" → select your repository
4. Leave all settings as default → click "Deploy"
5. Vercel will give you a live URL (e.g. harbour-hills-scripts.vercel.app)

## Run locally

```
npm install
npm start
```

Opens at http://localhost:3000
